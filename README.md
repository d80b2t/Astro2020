# Project Title

The working repo for the NEOCam Time Domain Extragalactic Science Astro2020 White Paper.


## Getting Started

A 'regular' git clone, e.g. 
> git clone https://github.com/d80b2t/Astro2020

will work just fine. 


### Prerequisites

LaTeX  
pdTeX
make

```
> make
```
